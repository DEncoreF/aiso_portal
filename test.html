Core Mechanisms

Four innovations that make it possible

01

Transparent Hooking

aiDAPTIV™ intercepts PyTorch's memory allocator at the lowest level. Applications require zero code changes — the system seamlessly redirects allocations to the SSD tier without any application-level intervention.

02

Predictive Pre-fetching

aiDAPTIVLink accurately predicts and streams the next required model slices from the SSD into VRAM while the GPU computes active tensors. This just-in-time data swap effectively prevents GPU starvation.

03

KV Cache Optimization

For long-context inference, computed KV Cache states are persisted to SSD. Subsequent queries retrieve these with O(1) complexity — eliminating O(N²) redundant computation and delivering up to 10× speed improvement.

04

Hardware Co-Design

Standard SSDs degrade quickly under continuous AI tensor workloads. Phison engineered the Flash Translation Layer (FTL) from scratch to optimise garbage collection and wear-levelling specifically for tensor movement patterns.
